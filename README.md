# pokemonMelody-arduino
Pokemon dual tone melody player using ATmega32U4 (Arduino Leonardo / Micro / Pro Micro)

## Pin Configuration
| name            | port | func | pin |
|-----------------|------|------|-----|
| Speaker (Ch0) + | PB5  | OC1A | 9   |
| Speaker (Ch0) - | PB4  | GPIO | 8   |
| Speaker (Ch1) + | PC6  | OC3A | 5   |
| Speaker (Ch1) - | PD4  | GPIO | 4   |
